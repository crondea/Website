Introduction

The menu bar above can be used to navigate the guide. If you have never used VirtualBox before, click Getting Started to begin! If you have used it before and are looking for a specific topic, feel free to skip around until you find it.

VirtualBox is a commonly used program for running virtual machines on your main operating system. Similar software exists, but the nice thing about VirtualBox is that it's free! This website is inteded to serve as a basic user guide on using VirtualBox. This guide will not cover every aspect and feature of the software, so other research may be required.
